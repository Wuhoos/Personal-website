NetFlix Clone

The "Netflix Clone" project is a React.js application styled with Tailwind CSS that replicates the basic functionality of Netflix. It features a home page that displays a random movie, a list of all available movies, and an option for users to add new movies. The app utilizes a JSON server to manage its movie library. This project provides a streamlined and interactive way to explore a small collection of movies, making it an educational example for those learning React and front-end development.
